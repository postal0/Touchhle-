- 🚀 Hızlı ve hafif Android emülasyonu
- 📱 Gerçekçi Android ortamı simülasyonu
- 🛠️ Geliştiriciler için gelişmiş araçlar
- ⚙️ Özelleştirilebilir ayarlar ve konfigürasyon
- 🔧 Kolay kurulum ve kullanım

## 📥 Kurulum

### Gereksinimler
- Android SDK
- Java Development Kit (JDK)
- Minimum 2GB RAM

### Kurulum Adımları

1. Repository'yi klonlayın:
```bash
git clone https://github.com/postal0/Touchhle-.git
cd Touchhle-
```

2. Gerekli bağımlılıkları yükleyin:
```bash
./install.sh
```

3. Emülatörü başlatın:
```bash
./touchhle
```
